# Composition-Workspace-Coordination-Console :-


The Composition Workspace & Coordination Console is a web-based document management and collaboration system designed to simplify content creation and project coordination. It allows users to create, edit, version, and review documents while managing tasks through a centralized coordination dashboard. The system combines document version control, commenting, workflow management, and task tracking into a single platform, improving collaboration and productivity for teams.


# Objective :-


1. Develop a collaborative document management system.
2. Provide automatic version control for documents.
3. Enable efficient task assignment and project coordination.
4. Improve communication through document comments and activity tracking.
5. Monitor project progress using an analytics dashboard.

# Key Features :- 


1. Document Management – Create, edit, delete, and organize documents.
2. Version Control – Automatically save document versions and restore previous versions.
3. Comment System – Add, resolve, and manage document comments for collaboration.
4. Task Coordination Console – Assign tasks, update task status, and track project progress.
5. Dashboard Analytics – View document statistics, task summaries, overdue tasks, collaborators, and activity logs.


# Technology Used :-

1. Frontend - HTML, CSS, Javascript
2. Backend - FastApi
3. Database - SQLite


# Result :-


1. Successfully developed a web-based collaboration platform.
2. Users can manage documents with automatic version history.
3. Team members can communicate through comments.
4. Tasks can be assigned and tracked efficiently.
5. Dashboard provides real-time project insights and activity monitoring.


# Conclusion :-


The Composition Workspace & Coordination Console provides an effective solution for collaborative document editing and project coordination. By integrating document versioning, task management, commenting, and analytics into one application, the system improves workflow efficiency, enhances collaboration, and ensures better project tracking. The modular FastAPI architecture also makes the project scalable and easy to maintain.

# How to run :-

Step 1 - 

Install Dependencies - pip install -r requirements.txt

Step 2 - 

Start the Server - uvicorn app.main:app --reload --port 8000
